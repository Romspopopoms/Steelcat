# SteelCat E-Commerce

Site e-commerce premium pour la litière SteelCat - Design épuré noir et blanc.

## Démarrage Rapide

### Installation
```bash
npm install
```

### Développement
```bash
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

### Build Production
```bash
npm run build
npm run start
```

## Stack Technique

- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Structure**: Architecture src/ avec App Router

## Structure du Projet

```
steelcat-ecommerce/
├── src/
│   ├── app/              # Pages et routing
│   └── components/       # Composants réutilisables
├── public/              # Assets statiques (images, logo)
└── ROADMAP.md          # Documentation complète et roadmap
```
